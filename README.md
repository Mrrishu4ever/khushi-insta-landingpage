## ✨ About the Project

A stunning personal portfolio and influencer landing page built for **Khushi Pawar** — a digital creator from Bangalore specializing in Fashion, Beauty, and Lifestyle content.

---

## 🎨 Features

- ⚡ **Animated Hero Section** — Bold typography with smooth entrance animations
- 🌙 **Dark Aesthetic Theme** — Deep purple/black with pink accent colors
- 🖱️ **Custom Cursor** — Pink dot cursor with trailing effect
- 📱 **Fully Responsive** — Mobile, tablet, and desktop optimized
- 🤖 **Khushi AI Chatbot** — Interactive AI twin for collab inquiries
- 🎞️ **Gallery Section** — Masonry-style photo grid
- 🗓️ **Journey Timeline** — Storytelling milestones
- 💌 **Collab/Contact Section** — PR, Brand Partnerships, Content Creation
- 🔠 **Marquee Ticker** — Scrolling text animation
- 🎬 **Loader Animation** — Branded loading screen

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/mrrishu4ever/khushi-insta-landingpage.git

# Open in browser
cd khushi-insta-landingpage
open index.html
```

No dependencies — pure HTML/CSS/JS! 🎉
